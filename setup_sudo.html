Установка sudo

  • Инструкция по установке
  • Назад к оглавлению

Sudo — очень гибкий инструмент, позволяющий настроить права на выполнения административных действий для каждого пользователя отдельно.
Например одному разрешить перезагружать какой-либо сервер, а другому дать возможность менять права доступа к файлам и папкам.
Для установки пакета sudo необходимы root-права.

В строке терминала ввести следующее:

>su –
>[ввести пароль пользователя root]
>apt install sudo
>[подтвердить установку нажав “Y”]
