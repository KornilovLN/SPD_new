Редактирование hosts

  • Инструкция по установке
  • Назад к оглавлению

Настройка файла, содержащего базу данных доменных имен и используемый при их трансляции в сетевые адреса узлов сохраняется в файле /etc/hosts.
Для редактирования файла необходимы root-права.

В строке терминала ввести следующее:

>su –
>[ввести пароль пользователя root]
>nano /etc/hosts

Файл hosts используется для локальной установки соответствия между IP адресом и доменом.
